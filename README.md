# Coddie-Desktop-A.I
This is a desktop assistant which can answer to most of your queries.

<h2>Pre-quesities needed:</h2>
 You need to install these modules via pip before running the script.

1:  pyttsx3
2:  speech_recognition
3:  requests
4:  subprocess
5:  wolframalpha
6:  pyautogui
7:  smtplib
8:  urllib
9:  BeautifulSoup(bs4)
10: googlesearch
11: selenium
